    Definitions of each selector

    Topic
    Refers to the 17 different groups for chronic disease indicators, as established by the CDC. Topic names can be found in the raw data under the "Topic" column.
    Question
    Refers to the various indicator names that describe the indicator value being measured (e.g., Cancer of the bladder, mortality). Question names can be found in the raw data under the "Question" column.
    Indicator
    Refers to the various indicator measures that are used for each indicator name (e.g., Crude Prevalence). Indicator measures can be found in the raw data under the "DataValueType" column.
    Year
    Refers to the year that the data for each indicator was collected. Data for the years 2007 to 2016 are included in the database. Year can be found in the raw data under both the "YearStart" and "YearEnd" columns.
    Location
    Refers to the location in the United States that the data was collected from. The granularity of the data is at the top-level administrative subdivision (i.e., 50 states, District of Columbia, and various territories). The raw data contains columsn for Location name ("LocationDesc"), Location abbreviation ("LocationAbbr"), and Location id ("LocationID"), the latter of which are numeric identifiers that were curated by the CDC.
    Population
    Refers to the race/ethnicity or gender of the individuals in a particular location that were sampled by each data source. Population can be found in the raw data under the "Stratification1" column.

    Descriptions of each query

    National Avg
    Allows users to construct a query for indicator names (i.e., Questions) and their respective locations in which the measure for that indicator is above or below the national average for that year. Users select the topic, question, indicator measure, year, and "above" or "below." Query returns
    Time
    Allows users to construct a query for states that had an increase or decrease in a particular indicator measure between two specified years. Users select the topic, question, indicator measure, year start, year end, and "increase" or "decrease." Query returns the years selected and their corresponding values along with the location.
    Top 10
    Allows users to construct a query for the top 10 locations with either the highest or the lowest value for a specified indicator measure for a specified population. Users select the topic, question, indicator measure, year, and population. Query returns the location and the corresponding value.
    Indicators
    Allows users to construct a query for all indicator names that had an increase or decrease in their yearly average between two specified years. Users select the topic, indicator, year start, year end, and "increase" or "decrease." Query returns questions along with their corresponding values for the two years selected.
    Location
    Allows users to construct a query for all locations and years that had the largest increase or decrease in a particular indicator name and measure. Users select the topic, question, indicator, and "increase" or "decrease." Query returns the location, years selected, and the percent difference in the measures between the two years.
